# Richman_swordland

## 遊玩方法
* 新遊戲 -> 可以選擇遊玩人數與遊玩角色
* 繼續遊戲 -> 每一個round中會遊戲會自動保存檔案, 若曾經有保存過檔案會直接運行
* 蓋房子讓其他人爆踩收稅以獲得最終勝利

## 操作方法
* 上、下、左、右、空白鍵、ENTER... 等常見按鈕

## 起始角色
* Kirito : 骰到高點數的機會較高
* Asuna : 不會骰到 1, 6
* Klein : 不會骰到 3, 4
* Heathcliff : 只會骰到 1, 5, 6

## 加入公會
* 血盟 : 對非成員收 1.25 倍的租金
* 微笑棺木 : 對所有人收 1.35 倍的租金
